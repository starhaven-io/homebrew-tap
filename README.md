# homebrew-tap

The official [Homebrew](https://brew.sh) tap for [starhaven.io](https://starhaven.io) software.

## Usage

```
brew tap starhaven-io/tap
brew install <cask-name>
```

## Casks

| Cask | Description |
| --- | --- |
| [`pinprick`](https://github.com/starhaven-io/pinprick) | Pin your GitHub Actions and prick holes in their supply chain security |
